# Huffman Desktop

A desktop application for the Huffman Archive tool, built using Electron.

## Overview
This application provides a user-friendly graphical interface for compressing and extracting files using the Huffman coding algorithm.

## Getting Started

### Prerequisites
- Node.js and npm installed on your system.

### Installation

1. Clone the repository:
   ```bash
   git clone https://github.com/Gokulovasan/huffman-desktop.git
   cd huffman-desktop
   ```

2. Install dependencies:
   ```bash
   npm install
   ```

3. Start the application:
   ```bash
   npm start
   ```

## Scripts

- `npm start`: Starts the application in development mode.
- `npm run make`: Builds and packages the application for your operating system.
